## 字典的遍历

```python
person = {'name': 'zhangsan', 'age': 18, 'gender': 'male'}

# 使用for...in..循环遍历一个字典，默认会拿到所有的key
for x in person:
    print('%s的值是%s' % (x, person[x]))

# 使用enumerate得到的是x的下标，y是key
for x, y in enumerate(person):
    print('%s的值是%s' % (x, y))
# 拿到所有的键值对
for x, y in person.items():
    print(x, y)

# 拿到所有的value
for y in person.values():
    print(y)

```

## 一、列表的格式

定义列的格式：`[元素1, 元素2, 元素3, ..., 元素n]`

变量tmp的类型为列表

```python
tmp = ['xiaoWang',180, 65.0]
Copy
```

**列表中的元素可以是不同类型的**

## 二、使用下标获取列表元素

```python
namesList = ['xiaoWang','xiaoZhang','xiaoHua']
print(namesList[0])
print(namesList[1])
print(namesList[2])
Copy
```

结果：

```python
xiaoWang
xiaoZhang
xiaoHua
```

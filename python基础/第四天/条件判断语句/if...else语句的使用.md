# if-else

## 一、if-else的使用格式

```python
    if 条件:
        满足条件时的操作
    else:
        不满足条件时的操作
```

+ 例如

  ```python 
  age = int(input('请输入您的年龄：'))
  # if判读条件：
  #   条件成立时执行的代码
  # else:
  #   条件不成立时执行的代码
  if age > 18:    
      print('欢迎来到我们的网吧')
  else:    
      print('请再等几年过来吧')
  ```

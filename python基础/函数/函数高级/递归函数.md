## 递归函数

### <1>什么是递归函数

一个函数可以调用其他函数。如果一个函数在内部不调用其它的函数，而是自己本身的话，这个函数就是递归函数。

**递归一定要留一个出口，不能一直无限的调用自己**

### <2>递归函数的作用

举个例子，我们来计算阶乘 `n! = 1 * 2 * 3 * ... * n`

解决办法1:使用循环来完成

```python
def cal(num):
    result,i = 1,1
    while i <= num:
        result *= i
        i+= 1
    return result

print(cal(3))
```

##### 看阶乘的规律

```
1! = 1
2! = 2 × 1 = 2 × 1!
3! = 3 × 2 × 1 = 3 × 2!
4! = 4 × 3 × 2 × 1 = 4 × 3!
...
n! = n × (n-1)!
```

解决办法2:使用递归来实现

```python
def factorial(num):
    result = 1
    if num == 1:
        return 1
    result = num * factorial(num -1)
    return result
print(cal(3))
```

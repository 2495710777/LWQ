.html .htm一样

<p>大吉大利，今晚吃鸡！</p>  段落标签两个段落之间中间空一行
<p>面对疾风咯！</p>
<font>大吉大利，今晚吃鸡！</font>  不能在body中直接打字，而是应该放到标签里面
<br/>   换行
<font>面对疾风咯！</font>
<hr color="green" width="100" size="100"/>   分割线
<!--空格没有用-->
<b>   你好啊   老铁！   </b>
<!--预排版标记 原样输出 标签照常输出-->
<pre>
    你好啊！老姜      <b>sb</b>饿了么
</pre>

<div>和<span>是两个最没有意义的标签，但是有时使用最多的两个标签

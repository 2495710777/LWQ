# 一.事件编程

## 1.绑定事件

#### bind(type,[data],fn);为元素绑定相关的事件处理程序

参数说明：

type:jquery事件类型

[data] :可选参数，事件发生时所传递的数据

fn: 完成事件的程序

```html
<script type="text/javascript" src="../jquery-1.8.3.js"></script>
<script type="text/javascript">
$(function(){
        $(":button").bind('click',function(){
                    alert('你好帅');
        });
         $(":button").bind('click',function(){
                    alert('真的！');
        });

});
</script>

<body>
    <input type="button" value="绑定事件"/>
</body>
```

### bind({type:fn,type:fn}):为元素绑定多个事件，json参数

```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<script type="text/javascript" src="jquery-1.8.3.js"></script>
		<script type="text/javascript">
			$(function(){
			        //给div绑定三个事件，一个是鼠标滑过，一个是鼠标离开，一个是单击事件
			        $("img").bind({'mouseover':function(){
			                               $('input').attr('value','你来了');
			            },'mouseout':function(){
			                               $('input').attr('value','你走了');
			            },'click':function(){
			                                $('input').attr('value','你敢点我');
			        }});
			});
		</script>
	</head>
	<body>
	    <input type="button" value="显示隐藏" id="btn"/><br/>
	    <img src="1.jpg" width="400" height="400"/>
	</body>
</html>

```

```html

                                                                            },'click':function(){
                                                                                $('input').attr('value','你敢点我');
                                                                            }});
```

## 1.基本动画

show(speed,[callback]):  显示   speed:速度,callback:事件完成时所触发的回调函数

hide(speed,[callback]) 同上

toggle() :切换显示或者隐藏（取反）

toggle(switch) : true : 显示   false:隐藏

toggle(speed,[callback])

speed:'slow','normal','fast'

```html
<script type="text/javascript" src="jquery-1.8.3.js"></script>
<script type="text/javascript">
$(function(){
        //给隐藏按钮绑定事件
        $(":button:eq(0)").bind('click',function(){
                $('img').hide(1000,function(){
					alert('我走了')
					});
		});
         //给显示按钮绑定事件
        $(":button:eq(1)").bind('click',function(){
                $('img').show(1000,function(){
					alert('我来了')
					});
        });
         //给切换显示隐藏按钮绑定事件
        $(":button:eq(2)").bind('click',function(){
                $('img').toggle(1500);
        });
});
</script>

<body>
    <input type="button" value="隐藏"/>
	<input type="button" value="显示"/>
    <input type="button" value="切换显示隐藏"/><hr/>
    <img src="1.jpg" width="400"/>
</body>
```

## 4.复制操作

clone():克隆元素

clone(true): 克隆元素同时复制元素的事件

```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<script src="jquery-1.8.3.js"></script>
		<script type="text/javascript">
			//定义一个定时器的名字
			var tm;
			$(function(){
				var inp= $('input')
				//点击input清除定时器
				$(inp).click(function(){
					clearInterval(tm)
				});
				//克隆一个具有事件的完整体(带了true)
				var clone_inp = inp.clone(true)
				//加在body里面的最后面
				$('body').append(clone_inp)
				$('p').css({border:'1px solid black'});
				//设置一个定时器
				tm = setInterval(function(){
					var text_=$('p').text();
					var one = text_.substr(0,1);
					var two = text_.substring(1)
					$('p').text(two+one)
				},100);
				console.log(tm)
			});
			
		</script>
	</head>
	<body>
		<p  >孩子，穷没事，还丑，还矮，还胖，还没内涵</p>
		<input type="submit" value="click" />
	</body>
</html>

```

## 7、查找操作

eq(index)：通过元素的索引查找元素，index默认从0开始

filter(expr)：查找与给定元素匹配的元素，expr匹配条件

not(expr)：查找与给定元素不匹配的元素

children([expr])：查找所有子元素

find(expr)：查找所有后代元素

next([expr])：查找紧邻的下一个元素

**nextAll();**查找紧邻的后面的所有元素

prev([expr])：查找紧邻的上一个元素

**prevAll();**查找紧邻的前面的所有元素

parent([expr])：查找当前元素的父元素

siblings([expr])：查找所有同级兄弟节点（无论上下）
